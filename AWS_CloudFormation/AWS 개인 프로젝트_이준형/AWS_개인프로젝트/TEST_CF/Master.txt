DB내의 기존 한글data도 utf8mb4로 변경해야함.

DB 접속 후

SHOW VARIABLES LIKE 'character_set%';
SHOW VARIABLES LIKE 'collation%';

위 명령어로 인코딩 방식 확인

아래 명령어로 기존 테이블의 인코딩 방식을 utf8mb4로 변경해야함

ALTER TABLE 테이블명 CONVERT TO CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

-- 1. Slave용 계정이 있는지 확인 (없다면 다시 생성)

-- 2. 복제 권한 다시 부여 (이게 핵심입니다!)
GRANT REPLICATION SLAVE ON *.* TO 'user1'@'192.168.111.200';
