mysql user1@'%' 사용자 생성 및 sqlDB 접속 권한 설정
CREATE USER 'user1'@'%' IDENTIFIED BY 'P@ssw0rd';
GRANT All privileges ON sqlDB.* TO user1@'%';
FLUSH PRIVILEGES;
========================================================
vim /etc/my.cfn 아래 내용 삽입

[mysqld]
collation-server=utf8mb4_general_ci
character-set-server=utf8mb4
validate_password.policy=LOW
default_authentication_plugin=mysql_native_password

[client]
default-character-set=utf8mb4

[mysql]
default-character-set=utf8mb4

systemctl restart mysqld

DB내의 기존 한글data도 utf8mb4로 변경해야함.

DB 접속 후

SHOW VARIABLES LIKE 'character_set%';
SHOW VARIABLES LIKE 'collation%';

위 명령어로 인코딩 방식 확인

아래 명령어로 기존 테이블의 인코딩 방식을 utf8mb4로 변경해야함

ALTER TABLE 테이블명 CONVERT TO CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
